## Pretrained Model

The trained YOLO model is hosted on Kaggle and is not included in this repository due to size limitations.

Download link:
https://www.kaggle.com/datasets/jasminesingh01/yolo11s-segmentation-model

Steps:
1. Download the model from the above-mentioned link.
2. Place `YOLO11s-seg_best.pt` inside the `models/` directory
3. Run inference scripts

Expected structure:
project-root/
│
├── models/
│   └── best.pt
